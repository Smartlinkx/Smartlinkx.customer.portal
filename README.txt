- js/portal.js
- css/style.css

SETUP
1. Upload all files to GitHub.
2. Edit assets/config.js
3. Replace API_BASE_URL with your Google Apps Script Web App URL.
4. Make sure your Apps Script backend already has these actions:
   - portalLogin
   - getPortalDashboard
5. Enable GitHub Pages.
6. Open portal-login.html from your GitHub Pages URL.

LOGIN
- Account No = subscriber account_no
- Password = subscriber portal_password
